# Guessing Number Game (Dart)

A simple command-line **number guessing game** written in **Dart**.
The program generates a random number and the player must guess it within a limited number of attempts.

---

## Project Description

This is a small console game built using Dart.
The computer randomly selects a number and the player tries to guess it.
After each guess, the program gives hints to help the player reach the correct number.

The game also includes difficulty levels and the ability to replay the game.

---

## Features

* Random number generation
* Three difficulty levels:

  * Easy (1 – 50)
  * Medium (1 – 100)
  * Hard (1 – 500)
* Limited number of attempts
* Smart hints:

  * Too High / Too Low
  * Very Close
  * Close
  * Far
* Play again option
* Simple command line interface

---

## Technologies Used

* **Dart**
* `dart:io` for user input/output
* `dart:math` for random number generation

---

## How the Game Works

1. The player chooses a difficulty level.
2. The program generates a random number within the selected range.
3. The player enters guesses.
4. The game provides hints after each guess.
5. The game ends when:

   * The player guesses the correct number.
   * The attempts run out.
