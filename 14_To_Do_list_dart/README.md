# To‑Do List Application (Dart Console App)

## About The Project

This is a simple **To‑Do List** application built using Dart in the terminal.
This project is great if you are:

* Learning Dart for the first time
* Preparing to start Flutter
* Improving coding skills

---

# Features

* Add new tasks
* View all tasks (numbered list)
* Edit existing tasks
* Delete tasks by number
* Exit the program gracefully

---

# Concepts Used

This project helps you practice important programming basics:

* Dart Lists
* Functions
* Switch‑Case
* Loops
* Input & Output
* Error Handling (`tryParse`)
* Clean Menu Navigation

These concepts are very important before starting **Flutter development**.
